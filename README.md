# Spam Detection

A simple spam detection project containing a dataset and a Python script.

## Files

- `spam.csv` - the dataset for spam classification.
- `tempCodeRunnerFile.python` - a Python script for working with the spam dataset.

## Usage

1. Open `tempCodeRunnerFile.python` in a Python editor.
2. Run the script to explore or build a spam detection model.
